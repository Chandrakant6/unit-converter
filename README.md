simplest unit converter

## supported units 
### time
~~~
seconds to decades
~~~
### length
### angle
### digital storage space
~~~
bit
...
terabyte
~~~
### temperature
~~~
celcius 
fahernehite
and kelvin
~~~
## run from CLI
syntax
~~~
python3 unit_convertor.py category value imitialUnit finalUnit
~~~
